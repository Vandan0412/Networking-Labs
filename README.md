# Networking Labs

A collection of hands-on networking labs completed as part of **CST8108 – Network Programming Basics** in the **Computer Engineering Technology – Computing Science** program at **Algonquin College**.

Each lab was completed in the college networking lab and then recreated in **Cisco Packet Tracer** to provide a reproducible environment for reviewing network topologies, configurations, and verification procedures.

This repository showcases practical networking experience and serves as a technical portfolio demonstrating foundational skills in network configuration, IP addressing, troubleshooting, connectivity testing, and traffic analysis.

---

## Lab Index

| Lab | Topic | Skills Demonstrated |
| :--- | :---- | :------------------ |
| **[Lab 01](01-LAN-Setup-Router-DHCP/)** | LAN Setup, Router Configuration & DHCP | Ethernet cabling, Cisco IOS CLI, DHCP configuration, IPv4 addressing, ICMP testing, connectivity verification |
| **[Lab 02](02-Client-Server-HTTP/)** | Client–Server Communication & HTTP Packet Analysis | Client–server networking, HTTP, TCP/IP, web server configuration, packet capture, protocol layer analysis |
| **[Lab 03](03-Router-Inter-Subnet-Routing/)** | Router Configuration & Inter-Subnet Routing | Router configuration, inter-subnet routing, Cisco IOS CLI, IPv4 subnetting, routing table analysis, gateway/ARP behaviour |
| **[Lab 04](04-IPv6-Addressing/)** | Configuring IPv6 Addressing | IPv6 addressing, IPv6 subnetting, Cisco IOS CLI, IPv6 routing, link-local addressing, NDP, connectivity verification |
